# Some copy and paste of projectoins used by the data processed here

## CARRA_Land_Pv2
This data uses the same projection as CARRA2

Polar stereographic projection (Can be south or north

```
lat0=90,
lon0=-30,
latc=84,
lonc=-45,
nx=2869,
ny=2869,
dx=2500,
```
